# UK-Bank-Customer-Insight-Dashboard - Power BI

## Project Overview
Interactive Power BI dashboard analyzing UK bank customer demographics, account balances, regional distribution, and customer trends through data visualization and business insights.

## Dataset Features
- Customer ID
- Customer Name
- Surname
- Gender
- Age
- Region
- Job Classification
- Date Joined
- Account Balance

## Key Insights & Visualizations

### Customer Demographics
- Region-wise Gender Distribution
- Customer Count by Gender
- Age Group Analysis

### Financial Analysis
- Average Balance by Job Classification
- Total Balance by Customer ID
- Balance Distribution Across Age Groups

- ### Time-Based Analysis
- Year-wise Customer Trends
- Quarterly Analysis
- Monthly Analysis
- Customer Join Trends

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Dashboard Features
- Interactive Filters and Slicers
- Dynamic Visualizations
- Customer Segmentation Analysis
- Financial Performance Insights

## Business Value
- Understand customer demographics and behavior.
- Identify regions with higher customer concentration.
- Analyze balance patterns across age groups and job classifications.
- Track customer growth trends over time.
